v1. Modos Principai
Tecla		Função
Esc		Volta para o Modo Normal (base de tudo).
i	        Entra no Modo de Inserção (antes do cursor)
a		Entra no Modo de Inserção (depois do cursor)
v		Entra no Modo Visual (seleção de caracteres).
V		Entra no Modo Visual de Linha (seleciona linhas inteiras).
Ctrl + v	Entra no Modo Visual em Bloco (seleção vertical).
:		Entra no Modo de Comando.

NAVEGAÇÃO
w / b		Próxima palavra (word) / Palavra anterior (back).
0 / $		Início da linha / Fim da linha.
gg		Vai para a primeira linha do arquivo.
G		Vai para a última linha do arquivo.
NUMBER + G	Vai para a linha específica (ex: 15G vai para a linha 15).
Ctrl + u / d	Sobe (up) / Desce (down) meia página.

EDIÇÃO BASICA
x		Deleta o caractere sob o cursor.
dd		Deleta (corta) a linha inteira.
yy		Copia (yank) a linha inteira.
p		Cola (paste) o conteúdo copiado/deletado após o cursor.
P		Cola antes do cursor.
u		Desfazer (undo).
Ctrl + r	Refazer (redo).
cw		Altera a palavra (change word) - apaga a palavra e entra em modo Insert.
r		Substitui um único caractere (sem entrar em modo Insert).

BUSCA E SUBSTITUIÇÃO
/texto			Busca por "texto" no arquivo (pressione Enter).
n / N			Próxima ocorrência / Ocorrência anterior.
*			Busca a palavra que está sob o cursor.
:%s/antigo/novo/g	Substitui todas as ocorrências de "antigo" por "novo" no arquivo todo.
:%s/antigo/novo/gc	Igual ao acima, mas pede confirmação (c) para cada troca.

ARQUIVOS E BUFFERS
:w			Salvar (write).
:q			Sair (quit).
:q!			Sair sem salvar (forçar).
:x ou :wq		Salvar e sair.
:e nome_arquivo		Abre um arquivo para edição.
:bn / :bp		Próximo buffer / Buffer anterior.
:bd			Fecha o buffer atual (mantendo a janela).
